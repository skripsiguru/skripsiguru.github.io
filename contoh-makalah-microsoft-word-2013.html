4. Jelaskan bagian bagian dari antarmuka microsoft word 2013


antarmukanya :
1. Window management, berisi tool untuk meminimalkan jendela, memaksimalkan jendela, menutup jendel, dsb.
2. Quick Access, berisi perintah cepat akses dokumen seperti undo, redo, table print dlsb.
3. Tab, terdiri dari tab FILE yang berisi perintah-perintah manajemen file. Tab HOME yang berisi perintah dasar teks editing. Tab INSERT terdiri dari perintah untuk menyisipkan sesuatu kedalam dokumen. Tab DESIGN yang berisi perintah untuk mengatur desain halaman. Tab PAGE LAYOUT yang berisi perintah-perintah untuk mengatur layout halaman. Tab REFERENCE berisi perintah-perintah yang berhubungan dengan referensi dokumen. Tab MAILINGS berisi perintah manajemen surat menyurat. Tab REVIEW berisi perintah-perintah yang berguna untuk proses kerja dan editing secara berkelompok dan tab VIEW yang berisi perintah untuk mengatur penampakan jendela kerja anda.
4. Title bar, menunjukan nama file dokumen yang sedang anda edit/kerjakan.
5. Help, jika anda kebingungan anda bisa mengakses menu ini dan meminta bantuan untuk masalah anda.
6. Ribbon Display Option, menu pilihan untuk memunculkan atau menyembunyikan Ribbon Interface.
7. Minimize, tombol untuk menyembunyikan jendela kerja Word 2013.
8. Maximize/Restore, tombol untuk memaksimalkan/meminimalkan ukuran jendela kerja Word 2013.
9. Close, tombol untuk menutup jendela kerja Word 2013.
10. Microsoft Account, menunjukan akun microsoft yang login pada Word 2013.
11. Ribbon Interface, adalah nama atau sebutan untuk gaya antar muka (user interface style) yang pertama kali diperkenalkan oleh microsoft pada Word 2007. Gaya ribbon interface ini tidak lagi menggunakan toolbar dengan dropdown menu yang memanjang kebawah. Menu-menu pada ribbon interface di urai ke samping dan menggunakan sistem tab sehingga mudah untuk diakses.
12. Ribbon command, tombol untuk menyembunyikan menu-menu perintah pada ribbon 13. Scroll dokumen, tombol ini bisa anda gunakan untuk menggerakan halaman dokumen ke atas atau ke bawah.
14. Zoom tools, gunakan tombol-tombol ini untuk memperbesar atau memperkecil penampakan dokumen.
15. Tombol view, gunakan tombol ini untuk mengubah mode tampilan dokumen, seperti mode baca, mode cetak, atau web layout 
16. Halaman kerja dokumen, tempat untuk melukiskan huruf demi huruf.
17. Informasi dokumen, berisi informasi jumlah kata dan jumlah halaman dokumen
18. Grup menu perintah, menu-menu yang memiliki tugas yang serupa akan dikelompokan ke dalam sebuah grup menu. Contohnya grup menu Font berisi menu-menu yang berhubungan dengan huruf atau font.

Penjelasan:

Perbedaan Microsoft Word 2013 dan 2015

Sebenarnya versi dari Microsoft word sangat banyak. Namun kita akan membahas yang versi baru saja. Yaitu Microsoft word 2013 dan 2015. Nah, ternyata walaupun jarak perilisan kedua versi Microsoft word ini tidak jauh beda, namun banyak perbedaan yang cukup mencolok. Apa saja itu? Berikut adalah perbedaannya Microsoft word 2013 dan 2015:

Media penggunaan aplikasi

Nah salah satu perbedaan microsoft word 2013 dan 2015 yang terlihat mencolok adalah media yang dapat digunakan untuk menjalankan aplikasi. Di Microsoft word 2013 penggunaan aplikasi hanya dapat diakukan melalui dekstop.

Lain halnya dengan Microsoft word 2015, kita dapat menggunakan aplikasi pengolah kata ini dimanapun. Baik itu dekstop, tablet bahkan smartphone. Sehingga jika kamu tidak terlalu suka membawa barang seperti laptop. Kamu dapat menggunakan Microsoft Word versi 2015 ini. Kamu dapat mengunduhnya di ponsel pintarmu.

 

Pengaturan untuk perangkat

Perbedaan microsoft word 2013 dan 2015 selanjutnya adalah untuk menyinkronkan pengaturan. Untuk penyingkronan perangkat telah disediakan di Microsoft word 2015. Sehingga untuk penggunaan perangkat yang berbeda, dapat disinkronkan.

Sinkronisasi yang dapat dilakukan melalui Microsoft Word 2015 adalah riwayat dokumen, tema, tata letak, dan masih banyak lagi.

Fitur membagikan dan menambahkan dokumen

Pada versi Microsoft word 2013 kita tidak dapat membagikan atau menambahkan sebuah dokumen. Namun untuk Microsoft word 2015 fitur ini telah disediakan. Jika kita membuat sebuah dokumen dan setelah selesai dokumen tersebut harus dibagikan kepada rekan kerja, atasan, atau orang yang memerlukan, fitur ini dapat digunakan sebagai solusi.

Selain itu kita juga dapat membagikan link ke sosial media. Sehingga orang lain dapat membuka dan mengedit nya. Dan juga jika kamu ingin mengubah format. Misalnya word ke pdf, word ke ppt, atau word ke txt kamu dapat langsung mengubahnya dalam satu aplikasi Microsoft Word 2015.

Kekuasaan untuk melakukan pengaturan

Perbedaan microsoft word 2013 dan 2015 adalah kekuasaan kita saat ingin melaikan pengaturan. Pengaturan disini ialah saat kita ingin melakukan kebebasan terhadap tampilan Microsoft word. Jika di Microsoft word 2013 kita memang dapat mengatur tampilan, namun tidak banyak dan leluasa.

Tapi di Microsoft word 2015 ini kita punya kesempatan tinggi untuk mengatur tampilan sesuka hati. Apakah kita ingin temanya berwarna putih, berwarna hitam, abu abu gelap, ataupun abu abu cerah. Kita juga dapat memilih sendiri layar belakang tampilan Microsoft word agar terlihat keren dan menarik.
