21. sebutkan dan jelaskan kelebihan Microsoft word dibandingkan dengan mesin ketik dan kelebihan Microsoft Excel dibandingkan dengan kalkulator!​


Jawaban:

Microsoft Excel atau Microsoft Office Excel adalah sebuah program aplikasi lembar kerja spreadsheet yang dibuat dan didistribusikan oleh Microsoft Corporation yang dapat dijalankan pada Microsoft Windows dan Mac OS.  

Aplikasi ini memiliki fitur kalkulasi dan pembuatan grafik yang, dengan menggunakan strategi marketing Microsoft yang agresif, menjadikan Microsoft Excel sebagai salah satu program komputer yang populer digunakan di dalam komputer mikro hingga saat ini. Bahkan, saat ini program ini merupakan program spreadsheet paling banyak digunakan oleh banyak pihak, baik di platform PC berbasis Windows maupun platform Macintosh berbasis Mac OS, semenjak versi 5.0 diterbitkan pada tahun 1993.  

Aplikasi ini merupakan bagian dari Microsoft Office System.  

Kelebihan Microsoft Excel sebagai dibandingkan program pengolahan angka yang lain adalah:

1. Tersedianya fitur untuk membuat grafik dari data.  

2. Kemampuan menyimpan data dalam jumlah besar dibanding dengan program lain.  

3. Rumus-rumus dan Formula pada Microsoft Excel lebih mudah dioperasikan.  

4. Adanya perbaikan integrasi antar aplikasi dalam Microsoft Excel.

5. File yang korup dalam Microsoft Excel lebih gampang untuk diselamatkan.

6. Terdapat berbagai tema yang akan mempermudah pengguna untuk memformat data.

7. Penulisan formula yang lebih mudah dibandingkan dengan program lain.

8. Adanya fungsi AutoComplete yang dapat menuliskan sintak-sintak rumus dengan tepat.

9. Tampilan grafik Microsoft Excel lebih memukau dengan efek-efek tiga dimensi, transparansi serta bayangan objek.

10. Terdapat file format Office Open XML.

11. Mempunyai kemampuan menampung data yang cukup besar, dengan baris dan kolom yang sangat banyak dalam satu (1) sheet, sehingga dalam satu (1) sheet bisa menampung jawaban dan pertanyaan yang lebih banyak pula.

12. Microsoft Excel mempunyai Format yang paling populer dan fleksibel jadi sebagian besar software data entry ada fasilitas konversi ke format excel atau format lain yang bisa dibaca excel. Atau jika dibutuhkan pengguna bisa konversi balik dari excel ke software statistik lainnya.  

13. Tersedianya fitur Pivot Table yang memudahkan pengguna bekerja lebih efektif karena semua tabel summary yang direncanakan bisa dibuat dahulu walaupun data belum masuk semua.

14. Setiap ada data masuk otomatis pivot table akan me-refresh sehingga tabel akan terupdate sendiri.

15. Kemudahan dalam menyoroti tren dan pola. Pengguna dapat mempermudah proses pemahaman tren dan pola dalam data Anda dengan bilah, warna, dan ikon untuk menyoroti nilai penting secara visual. Fitur prakiraan sekali klik baru dalam program terbaru menciptakan prakiraan di seri data pengguna dengan satu klik untuk penggunan terbaru.

16. Adanya fitur yang memungkinkan pengguna untuk bekerja sama secara real-time. Setelah lembar bentang disimpan ke OneDrive, OneDrive for Business, atau SharePoint, Anda dan tim dapat bekerja bersama secara real-time dengan Excel Online. Selagi pengguna melakukan pengeditan dan perubahan pada dokumen, riwayat versi di Excel terbaru yang telah disempurnakan memungkinkan Anda melihat atau kembali ke draf sebelumnya.

17. Adanya fitur berbagi dari awan. Setiap pengguna dapat memiliki versi buku kerja terbaru dengan membagikannya di awan menggunakan OneDrive atau SharePoint, sehingga orang lain dapat melihat, mengedit, dan bekerja sama.

18. Pengguna dapat menyesuaikan ritme terbaik untuk menggambarkan data dan ide secara visual. Ketika pengguna melihat pilihan yang paling tepat untuk menampilkan data, maka pengguna dapat menerapkan format, bagan mini, diagram, dan tabel dalam sekali klik. Serangkaian bagan dan grafik modern baru di Excel terbaru membantu pengguna menyajikan data dalam cara yang unik.

19. Mempermudah pengguna melakukan analisis. Program Microsoft Excel akan menjalankan analisis rumit dan meringkas data dengan pratinjau berbagai pilihan tabel pivot, sehingga Anda dapat membandingkannya dan menentukan salah satu pilihan yang paling sesuai dengan kebutuhan.

20. Format dan susun ulang data. Saat pengguna melihat konfigurasi yang berbeda, program Microsoft Excel akan mempelajari dan mengenali pola pengguna dan melengkapi otomatis data sisanya untuk pengguna. Tidak ada rumus atau makro yang diperlukan. Fitur pencarian ‘Beri Tahu Saya’ akan memandu pengguna pada perintah fitur yang pengguna perlukan untuk mendapatkan hasil yang pengguna cari.

21. Adanya fitur pengaturan tata letak data bagi pengguna. Pengguna dapat mengatur data numerik atau teks di lembar bentang atau buku kerja sehingga akan membantu pengguna dalam membuat keputusan berdasarkan informasi.

27. Tentang Microsoft Word Pengolah Kata​


Jawaban:

Microsoft Word merupakan program aplikasi pengolah kata (word processor) yang yang biasa digunakan untuk membuat laporan, membuat dokumen berbentuk surat kabar, membuat label surat, membuat table pada dokumen, dan masih banyak lagi dukumen- dokumen lain yang biasa dibuat dengan menggunakan Microsoft Word.

Jawaban:

Microsoft Word atau Microsoft Office Word atau Word adalah perangkat lunak pengolah kata (word processor) andalan Microsoft.Setelah diberi nama baru Microsoft Word, Microsoft menerbitkan program ini pada 25 Oktober 1983 untuk IBM PC. Saat itu dunia pengolah kata dikuasai oleh WordPerfect dan juga WordStar.
