16. Sebutkan menu yang ada pada menu tubulasi power point 2007


MENU PADA MICROSOFT POWERPOINT


MENU INSERT

KETERANGAN:

A. File/office buttom

B. save :untuk menyimpan file.

C.table : untuk membuat tabel.

D.picture : untuk memasukkan gambar.

E. clipart : untuk menambahkan clipart.

F. shape : untuk menambahkan bidang ruang/obyek.

G. grafik :untuk membuat grafik.

H. word art : untuk membuat text artistik.

I. symbol : untuk menambahkan simbol.

J. video :untuk menambahkan video.

K. audio : untuk menambahkan suara/lagu.

L. title bar/judul bar.


MENU HOME

KETERANGAN :

A. Save : untuk menyimpan dengan nama yang sama.

B. New slide : untuk menambah halaman baru.

C. Font : untuk mengedit font.

D. Layout : untuk mengatur tata halaman.

E. Paragraph : untuk mengatur paragraf.

F. Drawing : untuk membuat/mengatur obyek.


MENU DESIGN

KETERANGAN :

A. File/office buttom

B. Save : untuk menyimpan file dengan nama yang sama.

C. Undo : untuk mengembalikan kesebelumnya.

D. Theme : untukmengganti background halaman.

E. Background style : untuk mengatur background halaman.


MENU ANIMATION

KETERANGAN :

A. Save : untuk menyimpan file dengan nama yang sama.

B. Prevew : untuk melihat hasil.

C. Animation : untuk menambahkan animasi pada huruf/kata/kalimat/paragraf.

D. Advenced animation : untuk menambah animasi lain.

E. Timing : untuk memberi waktu kapan animasi akan di play dan berapa lama durasinya.


MENU TRANSITION

KETERANGAN :

A. Save : untuk menyimpan file dengan nama yang sama.

B. Prevew : untuk melihat hasil translate.

C. File/ office button

D. Transition to this slide : untuk menambahkan translate pada layout.

E. Timing : memberi waktu kapan akan ganti slide.


MENU SLIDE SHOW

KETERANGAN :

A. From beginning : untuk menjalankan slide mulai dari posisi awal hingga akhir slide

B. From current slide : untuk menjalankan slide mulai dari slide aktif atau slide yang sedang dipilih hingga akhir slide.

C. Custom slide show : untuk mengatur sendiri urutan slide presentasi sesuai dengan keinginan.

D. Set up slide show : untuk mengatur tampilan slide.

E. Use timings : untuk mengatur lamanya tampilan slide menggunakan waktu tertentu.


MENU REVIEW

KETERANGAN :

A. Spelling : untuk mengecek pengejaan kalimat.

B. Research : hampir sama dengan Find, namun ini lebih spesific.

C. Translate : untuk menerjemahkan ke bahasa lain.

D. Language : untuk memilih bahasa kalimat.

E. Comments : untuk memberikan catatan pada slide yang Anda buat.

29. Jelaskan apa yang di maksud ribbon pada Ms. Power point 2007 !


Ribbon adalah fasilitas berupa sekumpulan menu jalan pintas yang baru di Microsoft Office PowerPoint 2007. menggantikan sistem tab menu tradisional. Ribbon ini bisa berupa tab menu dan bisa juga berupa ikon. Hal ini dirancang agar responsif terhadap keinginan anda dan mudah digunakan bagian utama dari ribbon adalah Ribbon Tabs yang berisi sebanyak 7 menu utama yaitu : Home, Insert, Design, Animations, Slide Show,Review, dan View.

31. Untuk peretmuan hari ini silahkan dikerjakan soaltentang power point dibawa ini.1. Apa fungsi dari aplikai power point.?2. Sebutkan langkah untuk membuat slide powerpoint 2007?3. Sebutkan langkah untuk menyisipkan gambarpada power point 2007.?4. Sebutkan langkah untuk merubah backgroundslide power point 2007.?5. Sebutkan langkah untuk menyimpan slide powerpoint 2007.?​


Jawaban:

1.Fungsi dari Microsoft Office Power Point adalah untuk membuat dan mengolah/mengedit slide/file untuk presentasi. Selain itu, Power Point juga bisa untuk membuat animasi sederhana.

2.1. Masuk ke Power Point

2. Pada slide, pilih blank

3. Lalu buat slide yg akan di inginkan

4. Jika ingin membuat slide lagi,pilih new slide

5.Jika ingin membuat slide yg sama,pilih duplicate slide

3. 1.Klik tempat yang Anda inginkan untuk menyisipkan gambar di slide.

2.Pada tab sisipkan , di grup gambar , klik gambar , lalu klik perangkat ini.

3.Dalam kotak dialog yang terbuka, telusuri ke gambar yang ingin Anda sisipkan, klik gambar tersebut, lalu klik Sisipkan

4.1.Klik Tab Menu Design -> pilih background yang anda inginkan

2.Selanjutnya akan muncul tampilan slide dengan background yang sudah diubah

Cara lain untuk mengganti background slide melalui file dari komputer adalah sebagai berikut:

1.Lakukan hal yang sama seperti langkah pertama diatas.

2.Klik kanan pada area slide (kotak tulisan slide tidak aktif )

3.Klik Format Background

4.Klik Fill -> pilih opsi Picture or texture file

5.Klik File pada perintah Insert from

6.Pilih gambar mana dalam penyimpanan komputer anda untuk dijadikan background

7.Klik open untuk menampilkan background atau double klik pada gambar maka slide presentasi akan berubah backgroundnya.

5.Di panel gambar Mini slide, di sisi kiri jendela PowerPoint , pilih slide yang ingin Anda simpan. Klik File > Simpan sebagai (atau Simpan salinan jika presentasi disimpan di OneDrive atau SharePoint). Navigasikan ke folder tempat Anda ingin menyimpan slide.
