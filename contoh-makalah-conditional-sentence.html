3. 5 contoh kalimat future conditional sentence,present conditional sentence,and past conditional sentences


future/ type 3 :
If + subject + had + V3/past perfects + subject + would have + V3/past perfect
- If I had found her address, I would have sent her an invitation.
- If you had remembered to invite me, I would have attended your party.
- If he had asked you for forgiveness, would you have forgiven him?
- If he had called you, you could have gone.
- If I had had enough time, I would have come to your birthday party.

present/ type 1 :
If+ subject + VI (Simple Present Tense) + Subject + will + VI (Simple Present tense
- If I work hard, I will get a lot of money.
- If I have a lot of money, I will buy an expensive car.
- If I study hard, I will pass the final examination. 
- If I have free time, I will go swim at On The Rock Hotel.
- If I find her address, I’ll send her an invitation.

past/ type 2 :
If + subject + V2/simple past tense + Subject + would + Vl/past future tense
- If I were you, I would apply for that scholarship.
- If l got high a score in the final examination, my mother would give me sports car
- My husband would buy me a new car if he got a job in foreign company. 
- If I found her address, I would send her an invitation.
-  If I were you, I would not do this.

Conditional Sentences

----------------------------------------------------

Contoh kalimat pengandaian (conditional sentences)

Jawaban  I will go to school if my mother makes my breakfast. [Aku akan pergi ke sekolah bila ibuku membuat sarapan]  If she knows who he is, she will never love him anymore. [Bila dia tahu siapa dia, dia tidak akan pernah mencintainya lagi]  He would visit the zoo if he weren't ill. [Dia akan berkunjung ke kebun binatang seandainya dia tidak sakit]  If the house were bigger, they would like it. [Seandainya rumah tersebut lebih besar, mereka akan menyukainya]  You would have been mad if you hadn't known all about them already. [Kamu pasti akan marah seandainya kamu tidak mengetahui semua tentang mereka lebih dulu]  If I hadn't lied to them before, they would have believed me. [Seandainya aku tidak membohongi mereka, mereka pasti akan mempercayai aku]

--------------------

Penjelasan

Conditional sentences adalah kalimat yang berisi pengandaian. Conditional sentences pada umumnya terbagi menjadi 3, yaitu:

 Conditional Sentences Type 1 (First Conditional)  Conditional Sentences Type 2 (Second Conditional)  Conditional Sentences Type 3 (Third Conditional)

     Conditional sentences type 1 adalah tipe kalimat pengandaian yang masih memiliki kemungkinan untuk dapat terwujud. Conditional sentences type 2 adalah tipe kalimat pengandaian yang tidak nyata karena pengandaian di kalimatnya tidak terwujud pada masa kini. Conditional sentences type 3 adalah tipe kalimat pengandaian yang mustahil karena pengandaian di kalimatnya tidak terwujud pada masa lalu.

→  Rumus conditional sentences type 1 adalah sebagai berikut.

If + (simple present), Subject + will + Verb 1

atau

Subject + will + Verb 1 + If + (simple present)

→  Rumus conditional sentences type 2 adalah sebagai berikut.

If + (simple past), Subject + would + Verb 1

atau

Subject + would + Verb 1 + If + (simple past)

→  Rumus conditional sentences type 3 adalah sebagai berikut.

If + (past perfect), Subject + would + have + Verb 3

atau

Subject + would + have + Verb 3 + If + (past perfect)

ada 3 bentuk CS :

Type 1 : yang pasti terjadi (likely to happen), bentuk / struktur kalimatnya :
If + S + Verb 1 (Simple Present) + S + Future Tense (will + V1) + O
If + S + is / am / are (tobe) + adjective (kt.sifat) + S + Future Tense
Ex: If i have much money, i will buy a car
If i am rich, i will buy a motorcycle

Type 2 : diharapkan terjadi sekarang namun kenyataannya tidak terjadi, struktur kalimatnya:

If + S + Verb 2 (Past Tense) + s + Would + Verb 1

Ex : If i were a boy, i think i could understand
If only i could turn back time, i would stay for the night
If i were rich, i would buy a car

Type 3: semacam penyesalan (hal yang tak mungkin terjadi karena waktunya sudah lewat), bentuk kalimat :

If + S + past perfect + s + Past perfect future (would / could have + V3)
Ex : If Zakki had studied hard, he would have passed the test

28. contoh kalimat present conditional sentence​


║CONDITIONAL SENTENCE║

Answer

Present Conditional sentence is divided into two forms, the present real (truth) conditional and the  present unreal (untruth) conditional.

→ The present real conditional also called zero conditional refers to the habit or daily activities, something that we use to do in the reality.

Here is the pattern of the present real conditional :

If+Subject+verb 1(s/es) + Complement, S+verb 1 (s/es) + complement.

If + simple present , simple present.

Example

If there is nothing to do, I write the poems.

When my sister is tired, she often stays at home.

If I go to Yogyakarta, I usually stay at Jambuluwuk hotel.

Note : We use adverb "when" if the activity happens regularly. Meanwhile, we use "if" when the activity is less frequently.

→ The present unreal conditional also called the type 2 conditional refers to something that we can't reach now or just imagine of the particular condition.

Here is the pattern of the present unreal conditional :

If+Subject+verb 2 + Complement, S+would+verb 1 + complement.

If + simple past , past future

Example

If I were you, I would tell the truth to my mother.

If I had so much time, I would read novel.

If I went to Paris, I would spend all the money to buy souvenirs.

Explanation

Conditional sentence tells about the condition that will be actualized ( possible condition ) or only dreaming ( intention/Andai-andai ) because of the particular condition. Conditional sentence structure consists -If clause and result clause ( main clause ).

Use Comma when the result clause (main clause ) is started at the beginning sentence. Otherwise, there is no comma when -IF clause is put after the result clause.

→There are 4 types of Conditional Sentence←

1. ZERO CONDITIONAL

The Zero Conditional is used to describe the Phenomenon,  and General truth ( Fact  ). Zero Conditional uses the simple present tense form to reflect the truth.

Pattern :

IF+S+base verb (s/es ) + O/C , S+base verb (s/es ) + O/C

If + simple present , simple present

Example :

If we boil water , it evaporates up into atmosphere.

Analysis :

It tells about the general truth or phenomenon.

It uses the simple present tense form to describe the truth. It is proven that the water evaporates up and disappear when it is heated at 100 degrees Celsius.

2. TYPE 1 CONDITIONAL

The Type 1 Conditional is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened ). In fact, the speaker will actualize his/her dream now or soon .

Pattern :

If +S+ verb (s/es) + O, S+will+verb1+O/C

If + simple present, simple future

Example :

If I study hard, I will pass the exam easily.

Analysis

It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true. Meaning : it's possible that I will be successful for the exam later.

3. TYPE 2 CONDITIONAL

The Type 2 Conditional type  is used to describe  the untruth in the present / future. It is Contrary to the fact in the present or it could be in the futureThe speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future

"Keep in mind, if we use the first , second person and third person singular  ( I, she, he, it , ) in the nominal sentence , it must be followed by the PAST linking verb "were."" It is the same way to the plural form as the formal writing." If we use the PAST linking verb "was " , it should be used in the colloquial as the informal conversation.

Pattern:

If+S+verb2+O/C , S+would+verb1+O/C

If + simple past , past future

Example :

If I were a president, I would build the biggest library in Jakarta.

Analysis :

It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future. Meaning :  I can't build the biggest library because I'm not a president now.

4. TYPE 3 CONDITIONAL

The Type 3 Conditional  Is used to describe  the untruth in the pastIt is contrary to the fact in the past and It  reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb 3+O/C  , S+Would+have+Verb 3+O/C.

If + past perfect , past future perfect

Example :

If I had known my friend's house address correctly, I wouldn't have been lost.  

Meaning :  I didn't know my friend's house address so I was lost.

PENJELASAN

=> CONDITIONAL SENTENCE TYPE 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan terjadi di masa depan. Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang. Rumus dari kalimat ini adalah: if + simple present, simple future “will”.

=> CONDITIONAL SENTENCE TYPE 2

Second conditional sentences merupakan tipe kalimat pengandaian yang digunakan ketika hasil atau konsekuensi hanya memiliki harapan terwujud walaupun kemungkinannya sangat kecil. Dalam bentuk kalimat ini, rumus yang digunakan adalah if + simple past/were, would/could/might + bare infinitive. Penggunaan would/could/might berfungsi untuk menjelaskan seberapa besar sebuah kondisi itu akan terjadi.

=> CONDITIONAL SENTENCES TYPE 3

Third conditional sentence merupakan sebuah kalimat pengandaian ketika sebuah kondisi tidak mungkin terwujud sama sekali. Dalam tipe kalimat pengandaian yang ketiga ini, bentuk kalimat menggunaka past perfect yang dilengkapi dengan modal auxiliary seperti would, could, dan should. Rumus yang digunakan adalah if + past perfect, would/should/could/might + have + past participle.

33. Contoh kalimat interrogratif conditional sentence?


True in the Present or Future. (Benar pada waktu sekarang dan masa akan datang)
