4. contoh kalimat type 1 dirubah menjadi type 2 dan type 3


JAWABAN:
Conditional Sentence Type 1 :
If you study English seriously, you will pass the test.
(Bila kamu belajar Bahasa Inggris dengan serius, kamu akan lulus tes.)

Conditional Sentence Type 2 :
If you studied English seriously, you would pass the test.
(Seandainya dulu kamu belajar Bahasa Inggris dengan serius, kamu pasti lulus tes.)

Conditional Sentence Type 3:
If you had studied English seriously, you would have passed the test.
(Seandainya waktu itu kamu belajar Bahasa Inggris dengan serius, kamu pasti sudah lulus tes.)

PENJELASAN:
Kalimat pengandaian (Conditional Sentence) berarti kalimat yang menyatakan pengandaian.

Untuk C.S. type 1, kemungkinan pengandaiannya masih dapat terjadi, karena perwujudan pengandaiannya baru akan terbukti di masa depan.

Untuk C.S. type 2, kemungkinan pengandaiannya adalah tidak nyata, karena perwujudan pengandaiannya terjadi di masa kini (yang mana berbeda dengan kenyataan di masa kini).

Untuk C.S. type 3, kemungkinan pengandaiannya adalah mustahil, karena perwujudan pengandaiannya terjadi di masa lalu (dan tidak nyata karena tidak sesuai dengan kenyataan masa lalu).

5. Contoh kalimat conditional sentence type 1 diubah menjadi type 2


PART : -IF CONDITIONAL SENTENCE

-----------------

Jawaban:

TYPE 1

If i don't know the exact way, i will ask people on the street.

Fact : I probably ask people because i'm not sure that i can find the way properly. { Kemungkinan saya tanya orang karena tidak yakin bahwa asaya dapat menemukan jalan dengan tepat )

TYPE 2

If i didn't know the exact way, i would ask people on the street.

Fact : I know the exact way, so i don't need to ask people later . ( Saya tahu jalannya , jadi saya tidak perlu tanya orang saat ini dan ataupun mendatang )

TYPE 3

If i had not known the exact way, i would have asked people on the street.

Fact : I knew the exact way, so i didn't need to ask people yesterday. ( Saya telah mengetahui jalan yang tepat, jadi saya tidak perlu tanya orang kemarin )

Penjelasan:

CONDITIONAL SENTENCE/ ANDAI-ANDAItells about the condition that will be actualized ( possible condition ) or only dreaming ( intention ) because of the particular condition. Consists -If main clause and result clause.Using Comma if the -If clause is started at the beginning sentence.

Commonly, there are 4 types.

1. CONDITIONAL ZERO

is used to describe the  PHENOMENON and GENERAL TRUTH ( THE FACT )

Example :

If it rains, the streets get wet.

2. CONDITIONAL TYPE 1stating the truth  in the present/future ( The planning  will be possible / probably happened )the speaker will actualize his/her dream.using simple present tense and simple future

Pattern :

If +S+ verb (s/es) + O,  S+will+verb1+O/C

Example :

If i have some money, i will buy the English grammar book later.

Fact in the present/future : it's possible that i will buy the English grammar book today because i have enough money.

CONDITIONAL TYPE 2Stating the untruth in the present / futureContrary to the factThe speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/futureusing simple past and past futureKeep in mind, if we use the first and second person  ( I, she, he, it  ) , they must be followed by the linking verb "were." It is the same way to the plural form as the formal writing.

Pattern:

If+S+verb2+O....S+Would+verb1+O/C

Example :

If i were a president, i would build the largest library in the southeast Asia.

Fact in the present/future : I am not a president, i'm a student now .

3. CONDITIONAL TYPE 3Stating  the untruth in the pastContrary to the factusing the past perfect tense and past future perfect

Pattern :

If+S+Had+V3+O....,S+Would+have+V3

Example :

If my money hadn't lost , i would have bought the new cooking recipe.

Fact in the past : i lost my money, so i didn't buy the new cooking recipe yesterday

KESIMPULAN :

Conditional sentence sebagai kalimat andai-andai atau hanya impian jika persyaratan pendukung tidak tercapai.Dalam bahasa inggris terdapat 4 kategori kalimat pengandaian .Conditional Zero membahas tentang fakta/kejadian yang sebenarnya/fenomena alam. Bentuk strukturnya pada umumnya menggunakan simple present.Conditional type 1 membahas tentang prediksi/rencana nyata yang mungkin saja terjwujud karena persyaratannya direalisasikan saat ini atau mendatang. Conditional type 2 membahas tentang kejadian yang kontras ( berkebalikan ) dengan fakta nya yang terjadi saat ini atau mendatang. Dalam istilah ini, si pengucap hanya berandai-andai saja kerena persyaratannya tidak diwujudkan untuk saat ini atau nanti-nanti . Conditional type 3 membahas tentang kejadian yang kontras ( berkebalikan ) dengan faktanya yang terjadi di waktu lampau. Si pengucap mengulas kembali kejadian yang tidak berhasil terwujud di waktu lampau.

8. perbedaan Conditional sentences type 0 and type 1? berilah rumus dan contoh nya​


Jawaban:

Conditional sentence type 0

Tipe ini biasanya disebut sebagai zero conditional sentences.

Tipe kalimat ini digunakan ketika hasil atau konsekuensi dari kondisi terwujud alias memaparkan sebuah kebeneran (general truth) dan fakta ilmiah.

Kalimat ini biasanya berbentuk present tense dengan rumus: if + simple present, simple present. Contoh kalimat (berdasarkan contoh dari grammarly.com):

If we burn paper, it becomes ash.

I feel sick if I eat too much.

If you don’t brush your teeth, you get cavities.

When people smoke cigarettes, their health suffers.

Conditional sentence type 1

First conditional sentence merupakan bentuk kalimat pengandaian yang digunakan ketika hasil atau konsekuensi memiliki kemungkinan untuk terjadi di masa depan.

Hal ini bisa terjadi karena masih ada kondisi realistik yang masuk akal untuk kemungkinan terjadi.

Maka dari itu, bentuk kalimat dari conditional sentence tipe pertama ini memiliki bentuk kalimat simple future alias kalimat yang akan datang.

Rumus dari kalimat ini adalah: if + simple present, simple future “will” / imperative dan contohnya adalah:

If I meet him, I will introduce myself.

I will cook dinner tonight if you clean the house.

If you rest, you will feel better.

If you set your mind to a goal, you will eventually achieve it.

If it rains, I will stay at home.

If he gives her chocolate, she will be happy.

If it doesn’t rain, we will go to the library.

Contoh conditional tenses type 1 dan type 2 :

Conditional Type 1

1. If you come late, you will get punishment.

2. If you see Mrs. Anah, will you give this letter?

3. I will pick you up if it doesn't rain.

Conditional Type 2

1. If I knew his house, I would visit him.

Pembahasan

CONDITIONAL SENTENCE

1. Second Conditional

First Conditional digunakan untuk pengandaian ke masa depan yang belum dan mungkin terjadi.

Pola kalimat: If + Present Tense, will + infinitive

Contoh: If you visit your grandma, she will be happy.

2. Second Conditional

Second Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin terjadi di masa sekarang. Dengan kata lain, sesuatu dapat terjadi saat ini jika sesuatu yang lain di masa lalu dilakukan.

Pola kalimat: If + Past Tense, would + infinitive

Contoh: If we took U turn, we would not get lost.

3. Third Conditional

Third Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin karena sudah tidak terjadi (di masa lalu) .

Pola kalimat: If + Past Perfect Tense, would have + past participle

Contoh: If you had warned me, I wouldn't have gotten an accident.

Catatan Penting!

Untuk Second Conditional, 'was' selalu digantikan 'were' untuk semua subjek. Contoh: If he were not your brother, I would not lend my motorbike to him.

22. contoh conditional type 1 2 dan 3​


JAWAB:

If Conditional (Conditional Sentence)

contoh conditional type 1 2 dan 3

Type 1

If you study hard for the test, you will pass it

Type 2

If you studied hard for the test, you would pass it

Type 3

If you had studied hard for the test, you would have passed it

PEMBAHASAN:

Conditional Sentence

if conditional sentence merupakansebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

Contoh:  

If I go to Palembang I will meet Sarah  If we have a yacht, we will sail the seven seas If he have more time, he will learn karate If they tell their father, he will be very angry She will spend a year in the USA if it is easier to get a green card If I live on a lonely island, I will run around naked all day

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

Contoh:  

If I were you, I would forgive him If we had a yacht, we would sail the seven seas If he had more time, he would learn karate If they told their father, he would be very angry She would spend a year in the USA if it were easier to get a green card If I lived on a lonely island, I would run around naked all day

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb 3

Contoh :  

If Mimi had done her homework, she would have not got a punishment. If we had had a yacht, we would have sailed the seven seas If he had had more time, he would have learnt karate If they had told their father, he would have been very angry She would have spent a year in the USA if it had been easier to get a green card If I had lived on a lonely island, I would have run around naked all day

PEMBAHASAN:

Conditional Sentence

if conditional sentence merupakansebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). Dalam kata lain jika kalimat utama didepan, maka anak kalimatnya berada pada kalimat kedua (belakang) dan sebaliknya.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

Contoh:  

If I go to Palembang I will meet Sarah  If we have a yacht, we will sail the seven seas If he have more time, he will learn karate If they tell their father, he will be very angry She will spend a year in the USA if it is easier to get a green card If I live on a lonely island, I will run around naked all day

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

Contoh:  

If I were you, I would forgive him If we had a yacht, we would sail the seven seas If he had more time, he would learn karate If they told their father, he would be very angry She would spend a year in the USA if it were easier to get a green card If I lived on a lonely island, I would run around naked all day

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb 3

Contoh :  

If Mimi had done her homework, she would have not got a punishment. If we had had a yacht, we would have sailed the seven seas If he had had more time, he would have learnt karate If they had told their father, he would have been very angry She would have spent a year in the USA if it had been easier to get a green card If I had lived on a lonely island, I would have run around naked all day
