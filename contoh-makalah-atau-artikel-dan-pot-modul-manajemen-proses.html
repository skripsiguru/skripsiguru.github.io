2. Contoh Artikel tentang manajemen file


File System atau "Manajemen File" adalah metode dan struktur data yang digunakan sistem operasi untuk mengatur dan mengorganisasi file pada disk atau partisi. File sistem (Manajemen File) juga dapat diartikan sebagai partisi atau disk yang digunakan untuk menyimpan file-file dalam cara tertentu. Cara memberi suatu file sistem ke dalam disk atau partisi dengan cara melakukan format.
File mempunyai sifat sebagai berikut :
1. Persistence : Informasi dapat bertahan meski proses yang membangkit-kannya berakhir atau catu daya dihilangkan. Dengan properti ini maka file dapat digunakan untuk hasil-hasil yang diperoleh dari suatu proses dapat digunakan di masa datang.
2. Size : File umumnya berukuran besar. Memungkinkan menyimpan infor¬masi yang sangat disimpan
3. Sharability : File dapat digunakan banyak proses mengakses informasi secara kongkruen.

B. SASARAN MANAJEMEN FILE
Pengelolaan file adalah kumpulan perangkat lunak sistem yang menyedia¬kan layanan berhubungan dengan penggunaan file ke pemakai dan / atau aplikasi. Biasanya cara pemakai atau aplikasi mengakses file adalah lewat sistem. Pemakai atau pemrogram tidak perlu mengembangkan perangkat lunak khusus untuk mengakses data di tiap aplikasi. Sistem pun menyediakan pengendalian terhadap aset penting ini. Sasaran sistem file adalah sebagai berikut :
1. Memenuhi kebutuhan manajemen data bagi pemakai
2. Menjamin data pada file adalah valid.
3. Optimasi kinerja
4. Menyediakan dukungan masukan/keluaran beragam tipe perangkat penyimpanan.
5. Meminimalkan atau mengeliminasi potensi kehilangan atau perusahaan data.
6. Menyediakan sekumpulan rutin interface masukan/keluaran.
7. Menyediakan dukungan masukan/keluaran banyak pemakai di sistem multiuser.

Artikel penelitian berupa artikel yang mana dibuat berisikan berbagai jurnal hingga kumpulan buku teori artikel dan juga ditulis dengan rapi tata cara penulisan ilmiah pada umumnya.Artikel kajian pustaka berupa artikel yang mana berisikan berbagai refrensi dari buku hingga karya ilmiah namun dikutip dalam penulisan proposal secara baik.Artikel makalah berupa artikel yang mana termasuk karya tulis dibuat oleh seseorang yang membahas topik tertentu dan ditulis sistematis hingga adanya analisis yang baik.

Pembahasan

Artikel menjadi sumber bacaan penting untuk manusia modern, berikut beberapa manfaat artikel:

Sarana menyampaikan ide. Ide bisa dituangkan dalam berbagai hal seperti gambar dan penulisan suatu karya ilmiah, penulisan tersebut bisa menghasilkan dalam bentuk artikel. Dalam artikel makalah berisikan karya tulis yang dibuat oleh seseorang dengan menuangkan idenya sehingga bisa lebih mengekspresikan dirinya. Maka dari hal ini pentingnya penuangan ide agar bisa menghasilkan karya-karya yang bermanfaat.Memberikan pengetahuan akademis kepada para pembaca. Para pembaca mendapatkan pengetahuan dari membaca berbagai hal, salah satunya membaca artikel. Maka dari ini artikel bermanfaat untuk banyak orang sebagai sumber bacaan yang kaya akan pengetahuan.Berpikir sistematis tertib serta teratur. Skripsi hingga bacaan lainnya memiliki peranan penting agar membuat kita menjadi lebih cerdas, maka dari hal ini dalam pembuatan sumber-sumber bacaan tersebut harus dilakukan dengan sebaik mungkin termasuk rapi dan teratur.Memahami tujuan penulis. Tujuan penulis penting untuk dipahami jika para pembaca tertarik, tujuan penulis akan memberikan berbagai manfaat hingga solusi suatu masalah.
