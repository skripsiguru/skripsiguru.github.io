Jawaban:

Class sederhananya adalah sebuah blueprint object, class secara umum dipakai untuk paradigma pemograman berbasis object

Berikut contoh class pada javascript :

1) basic class => beserta penjelasannya

// pembuatan class pada umumnya di awali dengan huruf kapital ( mengikuti konvensi penamaan ada umumnya

class Orang{

 // class field

 species = 'human';

 // setiap class memiliki constructor ( syntax mungkin berbeda pada setiap bahasa pemograman

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

}

// instansiasi sebuah class, class tidak dapat dipangil langsung karena dia merupakan sebuah blueprint

const polin = new Orang('polin', 19);

// output Object polin (instansi dari Orang)

console.log(polin);

2. class dengan method

class Orang{

 species = 'human';

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

 // sebuah fungsi didalam class bisa juga disebut method

 // method ini ada untuk setiap instansiasi dari class Orang

 greet() {

   return `Halo nama saya, ${this.name}`;

 }

}

const polin = new Orang('polin', 19);

// output "Halo nama saya, polin"

console.log(polin.greet());

3.class inheritance (penurunan class)

class Orang {

 species = 'human';

 constructor(name, age) {

   this.name = name;

   this.age = age;

 }

 greet() {

   return `Halo nama saya, ${this.name}`;

 }

}

// extends berarti semua method dan field dari Orang ada di Murid

class Murid extends Orang{

 perkenalanDiri() {

   return `nama saya, ${this.name} dan umur saya ${this.age}`;

 }

}

//instansiasi dari murid turunan dari orang

const popon = new Murid('popon', 19);

// output "nama saya popon dan umur saya 19"

console.log(popon.perkenalanDiri());

class pada umumna pembuatannya sama pada setiap bahasa pemograman seperti harus memiliki constructor atau memiliki method, tetapi mungkin hanya berbeda syntax
