6. Ujung pegas digantungi beban yang sama besar . jika konstanta pegas k1=k2=k3=k4=k maka perbandingan periode susunan seri dan paralel adalah

Tp : Ts = __

konstanta gaya susunan paralel
kp = 4k

konstanta gaya susunan seri
ks = k/4

Periode getaran
T = π √(m/k)
periode berbanding terbalik dengan akar konstanta gaya

Perbandingan periode
Ts / Tp = √kp / √ks
Ts / Tp = √(kp / ks)
Ts / Tp = √(4k / k/4)
Ts / Tp = √16
Ts / Tp = 4 : 1 <--- jawaban

saya anggap keempat pegas seluruhnya disusun seri dibandingksn dengan jika keempat pegas disisun paralel.

Pada bagian ini, saya coba bentuk umum menentukan PERBANDINGAN periode dari sistem n buah pegas yang seluruhnya disusun seri dengan n pegas yang seluruhnya disusun paralel.
Konstanta gaya masing² pegas k
Massa beban m

Konstanta susunan seri
kₛ = k/n

Konstanta gaya susunan paralel
kₚ = n k

Periode getaran pegas
T = 2π √(m/k)
T berbanding terbalik dengan √k

perbandingan periode
Tₛ / Tₚ = √kₚ / √kₛ
Tₛ / Tₚ = √(kₚ / kₛ)
Tₛ / Tₚ = √(nk / (k/n))
Tₛ / Tₚ = √n²
Tₛ / Tₚ = n


• perbandingan periode getaran sistem 4 pegas seri dengan sisyem 4 pegas paralel → n = 4
Tₛ : Tₚ = n : 1 = 4 : 1

• perbandingan periode getaran sistem 2 pegas seri dengan sisyem 2 pegas paralel → n = 2
Tₛ : Tₚ = n : 1 = 2 : 1
