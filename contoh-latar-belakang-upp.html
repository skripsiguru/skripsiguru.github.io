11. Sebutkan Macam Macam Teknik Menggantung Yang Dilakukan Pada Gerakan Pull Upp Bantu Plisssssss ​


Jawaban:

4.Kaki yang ke angkat di silangkan.

1. Reguler pull up

Reguler pull up adalah gerakan pull up standar yang paling sering dilakukan. Pada pull up posisi pergelangan tangan anda menghadap berlawanan dengan bar pull up dan kemudian naikkan badan anda dan turunkan kembali. Pull ini sangat fokus untuk melatih otot punggung, otot leher, otot bahu, dan lengan secara keseluruhan.

2.Chin up

Chin up adalah kebalikan dari pull up. Chin up lebih berfokus untuk melatih otot bicep dan punggung anda. Olahraga chin up ini pada umumnya lebih ringan daripda pull up. Oleh karena itu, apabila anda belum mampu melakukan pull up, cobalah mulai melakukan latihan chin up sebagai tahapan awal.

3. One hand pull up

One hand pull up adalah olahraga pull up yang jauh lebih menantang. Gerakan ini dilakukan dengan cara bergelantungan pada bar pull up hanya menggunakan satu tangan. One hand pull up bisa anda lakukan jika anda sudah terbiasa melakukan pull up dan chin up secara rutin.

4.Hanging leg raise pull up

Ini adalah jenis pull up yang agak berbeda, karena pull up ini fokus untuk melatih otot-otot perut anda. Gerakan ini dilakukan dengan posisi kedua kaki lurus kemudian naikkan kedu kaki anda hingga membentuk L. Lalu turunkan kaki anda dan ulangi gerakan tersebut. Gerakan ini bisa menjadi alternatif pengganti sit up.

5.Behind the neck pull up

Pull ini dilakukan dengan posisi kepala anda menunduk di depan bar, sehingga punggung dan leher anda berada di belakang bar pull up. Tujuan pull up ini fokus untuk menonjolkan upper back anda supaya terlihat jauh lebih berotot. Pull ini juga sangat berat dilakukan jika anda belum terbiasa. Ada baiknya anda mencoba beberapa gerakan pull up baru mengembangkan pull up jenis ini
