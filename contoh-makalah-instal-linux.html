10. apa perbedaan logical dan primary pada saat membuat partisi saat instal linux?


PARTISI PADA HARDDISK Jawaban

Apa perbedaan logical dan primary pada saat membuat partisi saat instal linux?

Perbedaan utamanya adalah partisi primary adalah tempat dimana Sistem Operasi disimpan. Sedangkan pada partisi logical biasanya tidak terdapat Sistem Operasi.

Pembahasan

Partisi

Partisi adalah sistem berkas atau bagian-bagian media penyimpanan data pada harddisk. Partisi juga dapat diartikan dengan proses membagi ruang kosong pada hardidisk, untuk dapat memberikan file system pada harddisk tersebut, sehingga dapat diisi dengan berbagai macam data.

Jenis – jenis Partisi

Partisi Primary. Partisi Primary adalah partisi utama yang pertama kali diakses oleh komputer saat melakukan booting. Sistem Operasi biasanya terdapat dalam partisi ini. Selain sistem operasi, partisi ini juga dapat menyimpan data – data pribadi. Namun, tidak disarankan untuk menyimpan data di partisi ini karena untuk mencegah terjadinya kehilangan data saat melakukan install ulang sistem operasi.

Partisi Extended. Partisi ini berfungsi untuk mengatasi keterbatasan pembagian harddisk. Partisi ini tidak dapat secara langsung menangani data. Untuk hal tersebut, kita harus membuat partisi logical terlebih dahulu.

Partisi Logical. Adalah partisi sampingan yang terdapat pada partisi Extended. Partisi ini dapat digunakan untuk menyimpan berbagai macam file atau data yang kita miliki. Dalam partisi ini tidak terdapat sistem operasi.

11. Bagaimana cara menginstal sistem operasi linux


Bagaimana cara menginstall sistem operasi Linux. Linux merupakan salah satu sistem operasi yang populer sekarang. Macam-macam linux disebut dengan distro. Pilihlah distro yang sudak cukup terkenal dan terdapat dukungan dari pembuat distro atau dari forum-forum distro tersebut.

Cara menginstaal linux tidak jauh berbeda dengan menginstall sistem operasi Windows. Langkah pertama memastikan partisi hardisk. Namun terdapat perbedaan pada tipe partisi, jika windows menggunakan tipe NTFS, pada linux menggunakan ext4.

Pada saat membuat partisi secara manual pada linux bisa menggunakan format sebagai berikut

1. Pilih / dibaca root untuk menginstall sistem linux, alokasikan sesuai kebutuhan
2. Pilih /home untuk letak partisi menyimpan data-data.
3. Swap ruang swap berfungsi sebagai backup dari RAM kita, namun pada perangkat komputer laptop sekarang RAM sudah memiliki kapasitas yang besar, jadi alokasikan swap sebesar 1-2 GB saja.

Selain menggunakan partisi manual, Linux juga menyediakan partisi secara otomatis dan semua pembagian hardisk akan dilakukan secara otomatis namun data yang ada pada hardisk sebelumnya juga akan terkena partisi dan terhapus.

16. Buat flowchart penginstalan Linux, untuk jenis Linux nya manjaro​


Jawaban:

Berikut adalah flowchart penginstalan Linux Manjaro:

Pastikan komputer Anda memiliki koneksi internet yang stabil.

Unduh image instalasi Manjaro dari situs resmi Manjaro.

Buat bootable media, seperti USB flash drive atau DVD, dengan menggunakan image instalasi Manjaro yang telah diunduh.

Masuk ke BIOS atau UEFI pada komputer Anda dan atur booting dari media yang telah dibuat.

Jalankan instalasi Manjaro dari media bootable.

Ikuti langkah-langkah yang ditunjukkan oleh instalasi untuk mengatur partisi disk, konfigurasi jaringan, dan pengaturan lainnya.

Tunggu hingga instalasi selesai dan restart komputer Anda.

Setelah komputer restart, Anda akan masuk ke desktop Manjaro yang baru terpasang.

Catatan: Flowchart ini merupakan contoh umum dari penginstalan Linux Manjaro, dan mungkin tidak sesuai dengan setiap konfigurasi hardware dan pengaturan unik yang mungkin Anda miliki. Jika Anda memiliki pertanyaan lebih lanjut tentang cara menginstal Manjaro, disarankan untuk membaca dokumentasi resmi Manjaro atau mencari bantuan dari komunitas Manjaro.

Berikut ini adalah contoh flowchart penginstalan Linux Manjaro:

1. Mulai

2. Siapkan media instalasi Linux Manjaro, seperti CD/DVD atau flash drive.

3. Masuk ke BIOS/UEFI pada komputer dan atur boot priority agar komputer dapat boot dari media instalasi yang telah disiapkan.

4. Setelah komputer boot dari media instalasi, akan muncul menu instalasi Linux Manjaro.

5. Pilih bahasa yang ingin digunakan pada proses instalasi.

6. Pilih opsi "Install Manjaro" untuk memulai proses instalasi.

7. Pilih lokasi dan zona waktu yang sesuai.

8. Pilih opsi partisi yang ingin digunakan untuk menginstal Linux Manjaro.

9. Atur konfigurasi partisi sesuai kebutuhan.

10. Masukkan informasi pengguna dan password 11. yang akan digunakan pada Linux Manjaro.

12. Tunggu hingga proses instalasi selesai.

13. Setelah proses instalasi selesai, restart komputer dan boot dari hard drive yang telah terinstall Linux Manjaro.

Selesai.

25. jelaskan langkah pangkah menginstal squid pada linux debian


Squid adalah sebuah daemon yang digunakan sebagai proxy server dan web cache. Squid memiliki banyak jenis penggunaan, mulai dari mempercepat server web dengan melakukan caching permintaan yang berulang-ulang, caching DNS. 

1. Lakukan update

apt-get update

2. Ambil scriptnya

wget –no-check-certificate https://raw.githubusercontent.com/ipoelhuda/Install-Squid3/master/setup-squid.sh

3. Eksekusi

bash setup-squid.sh

Silahkan masukkan port yang di inginkan dan Enter, ingat port yang di masukan  jangan sampai bentrok dengan service yg lain. Disini saya menggunakan port 80 untuk squid.

Jika ingin menambah port silahkan isi lagi, tekan n untuk melanjutkan dan tunggu sampai installasi selesai.

Proxy squid sudah siap di gunakan, selanjutnya kita akan melakukan limit to ip. Apa maksud dari limit to ip? adalah mengizinkan ip tertentu untuk bisa tersambung atau menggunakan squid kita. Rugi kan, misalnya anda menyewa suatu vps dan menginstall squid di server lalu di pake rame-rame oleh orang lain tanpa sepengetahuan anda? selain bandwidth cepat habis juga akan memakan resource ram yang besar.

nano /etc/squid3/squid.conf

Tambahkan tanda # sebelum tulisan http_access allow all agar config limit yang akan kita buat bisa berjalan.

Tambahkan acl di bagian paling bawah

acl april dst 123.456.78.90
acl april dst 111.222.333.33
acl april dst 12.345.56.789
http_access allow april

Simpan dan reboot vps anda.

Done.

5. Pilih create new hard disk jika belum ada hardisk yang kits buat di Vboxnya. Jika sudah ada pilih Use existing hard disk. Kemudian klik next.

6. Lalu pilih tipe Virtual yang akan digunakan, karena kita menggunakan Vbox maka pilih VDI. Kemudian klik next.

7. Pilih Dynamically alocated sehingga ukuran virtual disk file bisa diubah – ubah, jika ingin tetap pilih fixed size. Kemudian klik next.

8. Lalu tentukan besar ukuran file untuk OS linux di Vbox. Kemudian klik next.

9. Lalu klik Create untuk membuat Virtual Disk baru.

10. Lalu klik Create untuk membuat Virtual mesin baru.

13. Lalu klik gambar folder di sebelah kanan untuk memasukkan file iso linux yang akan diinstall. Setelah itu, klik next.
