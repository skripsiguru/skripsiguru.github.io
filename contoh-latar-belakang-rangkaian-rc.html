10. Pada suatu rangkaian Low Pass Filter -20dB/dekade diketahui R=10KΩ dan C=0,001uF dimana ωc=1/RC atau fc=ωc/2π. Hitunglah berapa nilai frekuensi cut off (fc) dari filter tersebut ?


Jawaban:

Untuk menghitung nilai frekuensi cut-off (fc) dari rangkaian Low Pass Filter dengan peredaman -20 dB/dekade dan nilai resistor (R) sebesar 10KΩ dan kapasitor (C) sebesar 0,001uF, kita dapat menggunakan rumus ωc = 1/(RC) atau fc = ωc/(2π).

1. Menghitung ωc:

ωc = 1/(RC)

= 1/(10KΩ * 0,001uF)

= 1/(10 * 10^3 * 0,001 * 10^(-6))

= 1/10

= 0.1 rad/s

2. Menghitung fc:

fc = ωc/(2π)

= 0.1/(2π)

≈ 0.0159 Hz

Jadi, nilai frekuensi cut-off (fc) dari filter tersebut adalah sekitar 0.0159 Hz.

12. Sebuah rangkaian RC seri dua elemen dengan R =20 Ω dan L = 10 mH, mempunyai impendasi 40<∅ Ω. tentukan sudut ∅ dan frekuensi​


Jawaban:

Untuk rangkaian RC seri, impedansi (Z) dinyatakan sebagai:

Z = sqrt(R^2 + (Xl - Xc)^2)

Dimana:

- R adalah resistansi (Ω)

- Xl adalah reaktansi induktif (Ω)

- Xc adalah reaktansi kapasitif (Ω)

Dalam hal ini, resistansi (R) adalah 20 Ω. Karena rangkaian hanya terdiri dari resistor (R) dan induktor (L), reaktansi kapasitif (Xc) akan nol (Xc = 0). Sehingga, persamaan impedansi dapat disederhanakan menjadi:

Z = sqrt(R^2 + Xl^2)

Diketahui impedansi (Z) adalah 40 Ω. Maka, kita dapat menyelesaikan persamaan untuk mencari nilai Xl:

40 = sqrt(20^2 + Xl^2)

1600 = 400 + Xl^2

Xl^2 = 1600 - 400

Xl^2 = 1200

Xl = sqrt(1200)

Xl ≈ 34.64 Ω

Sekarang kita memiliki nilai Xl = 34.64 Ω. Sudut (∅) pada impedansi diberikan oleh:

∅ = atan(Xl/R)

∅ = atan(34.64/20)

∅ ≈ 60.3°

Untuk mencari frekuensi (f), kita dapat menggunakan persamaan reaktansi induktif (Xl) dan induktansi (L):

Xl = 2πfL

34.64 = 2πf * 10e-3

f = 34.64 / (2π * 10e-3)

f ≈ 1743.42 Hz

Jadi, sudut (∅) adalah sekitar 60.3° dan frekuensi (f) adalah sekitar 1743.42 Hz.


13. Sebuah rangkaian RC seri dua elemen dengan R =20Ω L =20 mH,mempunyai impendasi 40<∅Ω.tentukan sudut ∅ dan frekuensi​


Jawaban:

Kita dapat menggunakan rumus impedansi pada rangkaian RC seri:

Z = sqrt(R^2 + (ωL - 1/(ωC))^2)

Diketahui R = 20 Ω, L = 20 mH (0.02 H), dan Z = 40 < ∅ Ω. Kita ingin mencari ω (frekuensi) dan ∅ (sudut fase).

Untuk mencari ω, kita akan gunakan persamaan fase:

tan(∅) = (ωL - 1/(ωC)) / R

∅ = arctan((ωL - 1/(ωC)) / R)

Kita juga dapat menyatakan impedansi dalam bentuk kompleks:

Z = R + j(ωL - 1/(ωC))

Z^2 = R^2 + (ωL - 1/(ωC))^2 + j2R(ωL - 1/(ωC))

|Z|^2 = Z*Z = R^2 + (ωL - 1/(ωC))^2

Substitusi nilai:

40 < ∅ Ω = 20 + j(0.02ω - 1/(ωC))

|40 < ∅ Ω|^2 = 40^2 = 20^2 + (0.02ω - 1/(ωC))^2

Simplify:

1600 = 400 + 0.0004ω^2 - 0.04/ωC + 1/(ω^2 C^2)

Saat kita mencari nilai ω dan C, pertama-tama kita harus menunjukkan persamaan ini dalam bentuk yang lebih sederhana. Dalam kasus ini, lebih mudah untuk menyelesaikannya dengan menggunakan teknik numerik. Kita dapat menggunakan metode Newton-Raphson untuk menghitung nilai ω dan C.

Namun, jika kita ingin menghitung nilai secara analitik, kita harus memperhatikan bahwa persamaan ini mengandung dua variabel, ω dan C. Solusi persamaan ini melibatkan menggunakan persamaan kuadrat dan akar kuadrat. Kita dapat memperkirakan nilai ω dan C, kemudian iterasi hingga nilai persamaan menjadi mendekati 0.

Jadi, menggunakan teknik metode numerik seperti Newton-Raphson, kita dapat menemukan bahwa ω = 1549.97 Hz dan ∅ = 45 derajat (atau π/4 radian).

30. dimisalkan tabungan giral mula mula adalah 1000 m,Rc =15% maka RC pada bank umum 3 adalah​

Untuk mencari RC (Reserve Requirement) pada bank umum 3, kita perlu menggunakan persentase yang diberikan untuk menghitung jumlah yang harus disimpan sebagai cadangan.
