21. perbedaan array satu dimensi, dua dimensi dan tiga dimensi​


Berikut adalah perbedaan dari array satu, dua, dan tiga dimensi:

Satu dimensi: jenis elemen array yang hanya terdiri dari satu kolom atau baris. Array satu dimensi hanya bisa diakses dengan program yang memiliki indeks tertentu.Dua dimensi: jenis elemen array yang terdiri dari 2 bentuk index array sehingga memiliki dua atau lebih kolom dengan banyak baris. Array dua dimensi akan digambarkan dalam bentuk matriks atau grid.Tiga dimensi: jenis elemen array yang memiliki kolom, baris, dan juga berukuran lebih banyak ketimbang array dua dimensi. Penomoran index pada array tiga dimensi menggunakan 3 buah angka.

Pembahasan

Array adalah salah satu istilah yang digunakan dalam dunia data di mana array adalah suatu larik yang berisikan data-data yang memiliki tipe atau jenis serupa. Array biasanya digunakan untuk menggambarkan bagaiman suatu data disimpan. Tujuan utama dari penggunaan array pada kumpulan data kita adalah untuk menyederhanakan cara mengumpulkan atau mengkategorisasikan data-data kita. Maka dari itu, penggunaan array bisa meningkatkan efektivitas dan juga efisiensi kita ketika mengumpulkan suatu data karena bisa menjadikan beberapa data menjadi satu larik saja. Ada beberapa jenis array yang bisa kita gunakan, yaitu array satu dimensi, dua dimensi, dan tiga dimensi.

26. penulisan untuk mengisi nilai data dalam array multi dimensi adalah​


Dalam bahasa pemograman Java + Dua dimensi

package multiarray;

import java.util.Arrays;

import java.util.Scanner;

//Import utilitas Arrays, dan Scanner

public class MultiArray {

public static void main(String[] args){  

         int X,Y;  

     //Inisialisasi variabel X & Y sebagai Integer

Scanner jumlah = new Scanner(System.in);

     //Inisialisasi variabel jumlah sebagai objek Scanner

       

      System.out.print("Masukkan Jumlah array X : ");

      X = jumlah.nextInt();

     //Assign variabel X dengan inputan Scanner

       

      System.out.print("Masukkan Jumlah array Y : ");

      Y = jumlah.nextInt();

     //Assign variabel Y dengan inputan Scanner

       

       char arr[][] = new char[Y][X];  

       //Inisialisasi Dua dimensi

     

      Scanner kata = new Scanner(System.in);

     //Inisialisasi variabel kata dengan Scanner

      char C;

     //Inisialisasi Variabel C sebagai char

       

   for (int i=0; i<arr.length; i++){

      //Perulangan mengikuti panjang array Dimensi kesatu

           for(int j=0; j<arr[i].length; j++){  

              //Perulangan mengikuti array Dimensi kedua

               

               System.out.print("Masukkan Kata-Kata ["+i+"]["+j+"] :");

               C = kata.next().charAt(0);

              //Assign Variabel C dengan inputan Scanner

               

               arr[i][j] = C;  

              //Re-Assign Variabel arr[index i][index j] dengan isi variabel C

           }

           System.out.println();

      //Memberikan Space

       }

       Arrays.stream(arr).forEach(System.out::println);

     //Menampilkan array sebagai STRING, Bukannya char

   }  

   }

Cara pendeklarasian array satu dimensi dan aray dua dimensi
Contoh kode untuk mendeklarasikan array satu dan dua dimensi dengan bahasa Java adalah:

1. Array satu dimensi:
int[] ezra = new int[3];
(akan membuat array 1 dimensi bertipe integer dan bernama ezra, dengan panjang array 3)

2. Array dua dimensi:
String[][] lazuardy = new String[4][5];
(akan membuat array 2 dimensi bertipe String dan bernama lazuardy, dengan panjang baris array 4, dan kolom array 5)
